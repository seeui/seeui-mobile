# Button API

属性 | 说明 | 类型 | 默认值
-----|-----|-----|------
type | 按钮类型，可选：`primary`（主按钮） `default`（默认按钮） | string | primary
size | 按钮大小，可选：`small`（小按钮） `large`（独行按钮） | string | large
disabled | 设置按钮可用状态 | boolean | `false`
onClick | `click` 事件的 handler | Function(e: Event) | -
className | 额外类名 | string | -
